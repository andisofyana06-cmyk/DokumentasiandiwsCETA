Setiap file yang kamu kirim di sini diteruskan langsung ke chat Telegram-mu lewat Bot API. Telegram menyimpan filenya — halaman ini cuma jadi pintu masuknya. Tidak ada database, tidak ada server, tidak ada biaya hosting.

Kunci akses

Belum punya bot? Chat @BotFather di Telegram, kirim /newbot, lalu salin tokennya. Untuk Chat ID, chat bot kamu dulu (apa saja), lalu buka https://api.telegram.org/bot<TOKEN>/getUpdates dan cari angka "chat":{"id":...}.
